# WhatsAppScript Automated

## Requirements

+ Python 3.x
+ python-pip
+ selenium

## Setting up the Project

1. Clone the app ``` git clone https://github.com/rohitsh16/WhatsAppPyScript.git ```
2. ```mkdir project && cd project```
3. ```virtualenv .```
4. ```source bin/activate```
5. ```pip install selenium```

## Usage

``` python schedule.py "Friend's Name" "message" "HH:MM DD.MM.YYY" ```

For instant message,

``` python schedule.py "Firend's Name" "message" ```
